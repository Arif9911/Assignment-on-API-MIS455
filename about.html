Assignment on API (Country Data)

The Country Data App is a powerful tool designed to provide comprehensive information about countries worldwide. With an intuitive interface and seamless user experience, this application allows users to effortlessly access a wealth of data about any country they desire.

Using the latest APIs and cutting-edge technology, the Country Data App retrieves up-to-date information on various aspects of each country, including population, capital, flag, and more. Whether you're a student conducting research, a traveler planning your next adventure, or simply curious about the world around you, this app is your ultimate companion for exploring the diverse and fascinating countries of our planet.
